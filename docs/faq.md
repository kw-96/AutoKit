# AutoKit 常见问题（FAQ）

## 1. 安装与环境
**Q1：需要什么环境才能运行AutoKit？**
A：需安装Node.js 16+、MongoDB 4+，建议使用最新版Chrome浏览器访问前端界面。

**Q2：如何安装依赖？**
A：在根目录下执行`npm install`，分别进入backend和frontend目录执行`npm install`。

## 2. 配置与密钥
**Q3：Figma API Key如何获取？**
A：登录Figma官网，进入Settings → Personal access tokens，生成并复制Token。

**Q4：API Key验证失败怎么办？**
A：请检查Key是否正确、未过期，或重新生成。注意不要有多余空格。

**Q5：Figma组件库ID如何获取？**
A：在Figma网页端打开组件库，URL中file/后的一串字符即为ID。

## 3. 功能使用
**Q6：如何生成设计规范JSON？**
A：在"设计规范系统"页面输入Figma组件库ID和API Key，点击生成即可。

**Q7：如何生成新组件/设计稿/前端代码？**
A：分别进入"组件生成"、"设计稿生成"、"前端代码生成"页面，按提示输入参数并点击生成。

**Q8：支持哪些前端框架？**
A：目前支持React、Vue，后续可扩展更多框架。

## 4. 数据安全
**Q9：我的API Key和数据安全吗？**
A：所有密钥仅存储于MongoDB，传输过程加密。请勿将密钥泄露给他人。

**Q10：如何备份和恢复数据？**
A：可定期导出MongoDB数据，或使用系统的配置导出功能备份MCP配置。

## 5. 前后端联调
**Q11：前端页面无法访问怎么办？**
A：请确认前端服务已启动（默认5173端口），如端口被占用可修改配置。

**Q12：后端API无法访问怎么办？**
A：请确认后端服务已启动（默认3001端口），并检查MongoDB连接状态。

## 6. 常见报错与解决方案
- **API Key无效/过期**：重新生成并填写最新Key
- **Figma组件库ID错误**：确认ID为真实组件库ID
- **MongoDB连接失败**：检查数据库服务是否启动，配置是否正确
- **生成超时/失败**：检查网络连接，或稍后重试
- **配置保存失败**：检查数据库连接或联系管理员
