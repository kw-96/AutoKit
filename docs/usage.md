# AutoKit 使用指引（usage.md）

## 1. 典型用户流程
1. 登录系统，进入"MCP配置管理"页面，添加Figma MCP配置（填写API Key并验证）
2. 进入"设计规范系统"，输入Figma组件库ID和API Key，点击生成，自动提取设计规范JSON
3. 进入"组件生成"页面，输入新组件需求（如名称、类型等），系统自动生成组件结构
4. 进入"设计稿生成"页面，输入页面需求描述，系统自动生成设计稿数据
5. 进入"前端代码生成"页面，输入组件名、props等参数，系统自动生成前端代码
6. 下载、预览或导出生成的设计规范、组件、设计稿、代码

## 2. 主要界面说明
- **MCP配置管理**：管理Figma、AI等平台API Key，支持添加、编辑、验证、删除配置
- **设计规范系统**：输入Figma组件库信息，自动生成并展示设计规范JSON，支持版本管理
- **组件生成**：输入新组件需求，自动生成组件结构、属性、变体及文档
- **设计稿生成**：输入页面需求，自动生成页面级设计稿数据，支持预览与导出
- **前端代码生成**：输入组件参数，自动生成React/Vue等前端代码，支持下载

## 3. 常见操作步骤
### 3.1 添加Figma MCP配置
1. 打开"MCP配置管理"页面
2. 点击"新增配置"按钮
3. 选择平台类型（如Figma），填写API Key
4. 点击"验证"按钮，验证通过后保存

### 3.2 生成设计规范JSON
1. 进入"设计规范系统"页面
2. 输入Figma组件库ID和API Key
3. 点击"生成规范"按钮
4. 查看并下载生成的JSON文件

### 3.3 生成新组件
1. 进入"组件生成"页面
2. 输入组件名称、类型、描述等
3. 点击"生成组件"按钮
4. 查看生成的组件结构和文档

### 3.4 生成设计稿
1. 进入"设计稿生成"页面
2. 输入页面需求描述
3. 点击"生成设计稿"按钮
4. 预览、下载或导出设计稿数据

### 3.5 生成前端代码
1. 进入"前端代码生成"页面
2. 输入组件名、props等参数
3. 选择目标框架（React/Vue等）
4. 点击"生成代码"按钮
5. 下载生成的代码文件

## 5. 常见报错与处理建议
- **API Key无效/验证失败**：请检查API Key是否正确、有效，或重新生成密钥
- **Figma组件库ID错误**：请确认输入的ID为Figma组件库的真实ID，可在Figma网页端获取
- **生成超时/失败**：请检查网络连接，或稍后重试。如持续失败请联系管理员
- **MongoDB连接失败**：请检查数据库服务是否启动，配置是否正确
- **前端页面无法访问**：请确认前端服务已启动，端口未被占用

如遇其他问题，请参考[常见问题 faq.md](./faq.md)或联系技术支持。
