# AutoKit 前端界面

本目录为AutoKit项目的前端界面，负责MCP配置、设计规范管理、组件生成、设计稿与代码生成等交互操作。

## 技术栈
- React + TypeScript（推荐Vite脚手架）
- Ant Design（可选）

## 启动方式
1. 安装依赖
   ```bash
   npm install
   ```
2. 启动开发环境
   ```bash
   npm run dev
   ```

## 目录结构
```
/src         # 主要业务代码
/public      # 静态资源
```

## 说明
- 入口文件：src/main.tsx
- 推荐使用Vite进行开发
